# FlipChecker

FlipChecker is a Next.js web app and Chrome extension that shows you what Facebook Marketplace items are worth on eBay, instantly.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

The app is deployed on [Vercel](https://vercel.com) at [flipchecker.io](https://flipchecker.io).
